# PYNQ Bare-metal Motor Control

## Description
This project is a **bare-metal system** on the PYNQ-Z2 board, designed to learn **embedded programming** and **hardware/software co-design**.  
The final goal is a real-time **DC motor control** using a control loop, starting from basic exercises like LED blink and hardware PWM.

## Project Objectives
- Learn **embedded C** and **ARM assembly**  
- Manage **GPIOs, timers, and interrupts**  
- Create a simple **hardware PWM** in the FPGA  
- Implement a **real-time control loop** for a DC motor  
- Document the full workflow for a university **portfolio project**
